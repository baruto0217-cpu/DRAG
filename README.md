# 薬剤逆引き警戒アプリ 🚨

> **お薬手帳を見たら → この病態を疑え**
>
> 救急救命士・ER看護師向け薬剤逆引き参照アプリ

---

## 概要

救急現場でお薬手帳を見た際に、**薬名 → 注意すべき病態・現場アクション**を即座に確認できるアプリです。  
また、**傷病者の既往歴（疾患名）→ 服用中の可能性がある薬・注意事項**を逆引きする機能も搭載しています。

---

## 機能

### 💊 薬名から検索
- 薬名・商品名・病態名でリアルタイム検索
- カテゴリフィルター（出血リスク / 血糖・代謝 / 循環器 / 神経・精神 / ステロイド / 腎・電解質 / 気道）
- 重要度バッジ（🚨最重要 / 重要 / 注意）
- 各薬剤の「背景疾患タグ」表示
- タップで詳細展開（疑うべき病態・現場アクション・覚えておくべきポイント）

### 🏥 疾患名から検索
- 疾患名で検索（心房細動・糖尿病・てんかん など）
- 14疾患収録
- 各疾患で「服用中の可能性がある薬」と「注意事項」を一覧表示

---

## 収録データ

| 種別 | 件数 |
|---|---|
| 薬剤アラート（薬名 → 病態） | 20パターン |
| 疾患逆引き（疾患名 → 薬） | 14疾患 |

### 薬剤アラートカテゴリ

- 🩸 出血リスク（ワルファリン・DOAC・抗血小板薬）
- 💉 血糖・代謝（インスリン・SU薬・SGLT2阻害薬・メトホルミン）
- 💓 循環器（ジゴキシン・β遮断薬・Ca拮抗薬・アミオダロン）
- 🧠 神経・精神（リーマス・抗精神病薬・TCA・デパケン・SSRI）
- 💊 ステロイド・免疫（副腎不全クリーゼ）
- ⚗️ 腎・電解質（利尿薬・ACE阻害薬）
- 🫁 気道・呼吸（テオフィリン）


## セットアップ

```bash
# リポジトリのクローン
git clone https://github.com/your-username/drug-alert-app.git
cd drug-alert-app

# 依存関係のインストール（React + Vite を推奨）
npm create vite@latest . -- --template react
npm install

# 開発サーバー起動
npm run dev
```

> **注意：** `src/` ディレクトリを Vite プロジェクトの `src/` に配置してください。  
> エントリポイント（`main.jsx`）は Vite のデフォルト設定をそのままご利用いただけます。
